📘 Основы программирования на Python

Три урока: переменные, условия, циклы — для начинающих школьников 🐍✨

📦 Урок 1. Переменные

Что такое переменная?

Представь, что у тебя есть коробка с наклейкой. На наклейке написано имя коробки (например, age), а внутри лежит какое-то значение (например, число 12). В программировании переменная — это именно такая именованная коробка для данных.

Компьютер выделяет для неё маленький кусочек памяти, и мы можем в любой момент положить туда новое значение или достать старое.

Создание переменной (присваивание)

Чтобы создать переменную, нужно написать её имя, поставить знак = (это оператор присваивания) и указать значение.

age = 12
⚡ Важно: знак = не означает «равно» в математическом смысле. Он означает «положи в переменную то, что справа».

Типы данных

В Python переменные могут хранить разные типы данных. Самые популярные:

  • int (целое число) — например, 10, -5, 42
  • str (строка, текст) — обязательно в кавычках: "Hello", 'Python'
  • bool (логическое значение) — только True (истина) или False (ложь)
name = "Olya"           # str
height = 150            # int
is_student = True       # bool

Обрати внимание: имена переменных чувствительны к регистру — name и Name — это разные переменные.

Как использовать переменные?

Переменные можно: выводить на экран, складывать/вычитать/умножать (если это числа), объединять строки, изменять присваиванием.

# Вывод
print(name)            # Olya

# Математика
petya_age = 10
masha_age = 12
total_age = petya_age + masha_age
print(total_age)       # 22

# Работа со строками
greeting = "Hello, " + name + "!"
print(greeting)        # Hello, Olya!

# Изменение значения
petya_age = 11         # теперь в переменной 11

Имена переменных: правила и советы

  • Имя должно быть осмысленным: apple_count лучше, чем x.
  • Состоит из букв, цифр и нижнего подчёркивания _.
  • Не может начинаться с цифры: 2variable — нельзя.
  • Нельзя использовать ключевые слова Python (например, if, for, while).
  • Если имя длинное, используйте _ для разделения слов: current_score.
  • Принято писать имена переменных строчными буквами (lowercase).

🚦 Урок 2. Условные конструкции

Зачем нужно условие?

В жизни мы постоянно принимаем решения: «если идёт дождь, то возьму зонт, иначе не возьму». В программировании то же самое — программа может выполнять разные действия в зависимости от данных.

Конструкция if (если)

Самая простая форма — if. Если условие истинно (равно True), код внутри выполняется; если ложно (False) — пропускается.

if condition:
    # этот код выполнится, если condition True

Важно: отступ (4 пробела или клавиша Tab) показывает, какой код относится к условию.

temperature = 25
if temperature > 20:
    print("It's warm outside! ☀️")
# Код после отступа закончился, дальше выполняется всегда
print("Check completed")

Конструкция if-else (если-иначе)

Если нужно выполнить одно действие при истинном условии, а другое — при ложном, используем else.

temperature = 10
if temperature > 15:
    print("You can go without a jacket. 🌞")
else:
    print("Better wear a jacket. 🧥")

Здесь условие temperature > 15 ложно, поэтому выполнится блок после else.

Конструкция if-elif-else (много вариантов)

Когда вариантов больше двух, используем elif (сокращение от «else if»). Проверки идут по порядку: как только какое-то условие оказывается истинным, его блок выполняется, а остальные пропускаются.

grade = 4

if grade == 5:
    print("Excellent! 🎉")
elif grade == 4:
    print("Good! 👍")
elif grade == 3:
    print("Could be better. 📚")
else:
    print("Need to improve. 💪")

Операторы сравнения

ОператорЗначениеПример (истина)
==равно5 == 5
!=не равно5 != 3
>больше10 > 5
<меньше3 < 7
>=больше или равно5 >= 5
<=меньше или равно4 <= 5

Получение данных от пользователя

Функция input() позволяет программе спросить что-то у пользователя. Она всегда возвращает строку (str). Если нужно число, используем int().

name = input("What's your name? ")
print("Hello, " + name + "!")

age = int(input("How old are you? "))
if age >= 18:
    print("You are an adult.")
else:
    print("You are still a school student.")

✏️ Задание

  1. Спрашивает у пользователя его возраст (input).
  2. Превращает ответ в число (int).
  3. С помощью if-elif-else выводит:
    • «You can watch cartoons» — если возраст меньше 12,
    • «You can watch educational videos» — если возраст от 12 до 17 (включительно),
    • «All materials are available» — если возраст 18 и больше.
  4. Если пользователь ввёл что-то не число, программа может выдать ошибку — пока мы это не обрабатываем.

🔁 Урок 3. Циклы

Зачем нужны циклы?

Представь, что нужно 100 раз написать «I love Python». Писать 100 строк print — долго и скучно. Циклы позволяют повторить блок кода нужное количество раз или пока выполняется какое-то условие.

Цикл for — повторение с известным числом шагов

Чаще всего for используют с функцией range(). range(n) генерирует числа от 0 до n-1.

for i in range(5):
    print("I love Python! 🐍")

Выполнится 5 раз. Переменная i будет последовательно принимать значения 0, 1, 2, 3, 4. Даже если мы не используем i внутри цикла, она всё равно нужна для счёта.

Если нужно начать не с 0, а с другого числа, пишем range(start, stop):

for counter in range(1, 6):
    print("Count:", counter)

Выведет:

Count: 1
Count: 2
Count: 3
Count: 4
Count: 5

Цикл for можно использовать не только с числами, но и для перебора элементов списка или символов строки — это мы изучим позже.

Цикл while — повторение, пока условие истинно

Цикл while проверяет условие перед каждым шагом. Если условие истинно — выполняет тело цикла, потом проверяет снова. Как только условие становится ложным, цикл заканчивается.

secret_word = "Python"
guess = ""

while guess != secret_word:
    guess = input("Guess the secret word: ")
    if guess != secret_word:
        print("Wrong! Try again.")

print("Congratulations! 🎉")

В этом примере цикл будет выполняться, пока пользователь не введёт слово «Python». Количество повторений заранее неизвестно, поэтому while здесь подходит идеально.

Опасность бесконечного цикла

Если условие в while никогда не станет ложным, цикл будет крутиться вечно. Программа «зависнет». Чтобы этого избежать, убедись, что внутри цикла что-то изменяет условие (например, пользователь вводит данные, или переменная-счётчик увеличивается).

# Пример бесконечного цикла (НЕ ЗАПУСКАЙ!)
x = 5
while x > 0:
    print("Infinite loop!")
# Нет изменения x, условие всегда True

Правильно будет добавить изменение переменной:

x = 5
while x > 0:
    print("Steps left:", x)
    x = x - 1   # уменьшаем x, чтобы условие стало ложным

✏️ Задание

  1. Спрашивает у пользователя, сколько раз поздороваться (используй int(input())).
  2. Если пользователь ввёл число больше 0, программа с помощью цикла for выводит столько раз «Hello, friend! 😊».
  3. После цикла выводит сообщение «Done!».
  4. Если пользователь ввёл 0 или отрицательное число, программа выводит «Invalid number» и завершается (без цикла).

🔍 Бонус: для самых любопытных

Попробуй объединить все три темы в одной программе. Например, напиши мини-игру:

  • Переменная хранит загаданное число.
  • Цикл while просит пользователя угадать.
  • Внутри цикла условие if проверяет, больше или меньше введённое число.
  • Когда угадал — выходим из цикла.

Так ты увидишь, как переменные, условия и циклы работают вместе! 😎
